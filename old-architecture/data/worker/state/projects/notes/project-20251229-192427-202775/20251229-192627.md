## 1) Layering blueprint (simple, proven stack)
**Goal:** stay warm while standing still for 1–3+ hours, minimal sweating.
1) **Base layer (next-to-skin):** moisture transport + some warmth  
2) **Midlayer (fleece/wool):** “working insulation,” easy to vent  
3) **High-loft insulation (puffy jacket/pants):** biggest warmth jump for static time

5) **Extremities:** insulated boots + thick socks; liners + mittens; balaclava/hat + gaiter

## 3) Sizing & fit rules (so the system actually works)
- **No compression in insulation zones:** toes, fingers, torso puffy, thighs. Loft = warmth.  
- **Boot fit:** thick socks + toe wiggle. If you can’t wiggle toes, you’ll get cold.  
- **Layer compatibility check (do this indoors):** put on all layers, then:
  - raise arms overhead, squat, walk stairs
  - zip everything fully and see if neck/waist drafts exist
- **Avoid overdressing to the point you sweat** before you settle into stillness. If you’re walking to the spot, start slightly cool, then add your puffy/parka at the viewing location.
