# Reflection Templates

Templates to help capture life data that enriches your profile over time.

## Weekly Reflection

**File**: `weekly_reflection.md`

**Purpose**: Captures work, health, relationships, finances, and learning in one pass. Fills the domain gaps that cause profile imbalance.

**How to use**:

1. Copy the template:
   ```bash
   cp weekly_reflection.md ../state/inbox/pending/reflection-$(date +%Y-%m-%d).md
   ```

2. Fill it out (5-10 minutes)

3. The Ingestion agent will process it automatically

**Recommended cadence**: Sunday evening or Monday morning

**What it feeds**:
- Temporal profiles (richer cross-domain data)
- Synthesis (better identity constraint evidence)
- World agent (more diverse opportunity matching)
- Attention (better energy/mood pattern recognition)

## Tips

- **Don't overthink it** — brief notes beat skipped weeks
- **Be honest** — "nothing" or "skipped" is valid data
- **Name people** — relationships need specificity to track
- **Include friction** — what drained you is as important as wins
